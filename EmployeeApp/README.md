# Employee App

This Expo app is the starting point for the employee mobile experience.

## Start development

```bash
npm install
npx expo start
```

The app uses file-based routing in the `app` directory.

## Useful commands

- `npm run android` to open Android
- `npm run ios` to open iOS
- `npm run web` to open the web build
- `npm run lint` to check code style
